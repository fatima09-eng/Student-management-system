### Student Management System in C++

This is a console-based student management system written in C++. It uses file handling to store and manage student data without using a database.

**Features:**
- Add new student records
- View all students
- Search student by ID
- Update student details
- Delete student record

**Tech Used:** C++, File Handling

Run the `SMS.cpp` file in any C++ compiler like Code::Blocks, Dev C++, or VS Code with MinGW.
